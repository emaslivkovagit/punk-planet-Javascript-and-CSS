Reflection on Recreating Page 40 Using HTML, CSS, and JavaScript

This project involved recreating page 40 of Punk Planet issue 62 in 5 files: HTML for structure (recreated page, index, and reflection), CSS for styling, and JavaScript for interactivity.

What Was Easy

Rewriting the HTML structure was easier. Once the semantic tags and class names were in place, linking the CSS and JavaScript files was quite simple. Setting global styles on the body element also saved a lot of repetition throughout the stylesheet.

What Was Challenging

Getting the three-column CSS Grid layout to feel like a real magazine page took fine-tuning of gap sizes, padding, and text alignment. It was also challenging to include all the necessary css parts into the sylesheet, as well as come up with some ideas for the JS part.

What Was Not Possible

Not including the illustration in the middle of the page. Making sure that all paragraphs begin and end the same as the original newspaper page 40.
